# Toward General-Purpose AI: Beyond Next-Token Prediction to Cognitive Architectures

This repository contains supplementary materials for the paper:

**Toward General-Purpose AI: Beyond Next-Token Prediction to Cognitive Architectures**

## Overview

This work surveys the current limits of foundation models and argues that the next major challenge for general-purpose AI is not scale alone, but the lack of explicit mechanisms for coordinating memory, reasoning, perception, action, and control over time.
